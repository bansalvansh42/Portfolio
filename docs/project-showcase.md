# Project Showcases

All projects follow a consistent narrative:
Problem → Solution → Tech Stack → Quantified Impact

## AI-Powered Voice Appointment Scheduler

Problem:
Businesses spend hours manually scheduling appointments via phone.

Solution:
Voice AI assistant that handles full-cycle appointment booking through natural phone conversations. Integrates with Google Calendar for real-time slot availability and confirmation.

Tech:
- VAPI (voice AI platform)
- GPT-4 (conversation orchestration)
- FastAPI (backend API)
- Google Calendar API (slot management)
- Twilio (telephony)

Impact:
70% reduction in manual scheduling effort. Handles 200+ appointments per week with 92% booking completion rate.

## IT Helpdesk AI Assistant

Problem:
Tier-1 IT support tickets overwhelm helpdesk staff with repetitive questions.

Solution:
RAG-powered assistant over internal knowledge bases with smart ticket escalation.

Tech:
- LangChain (agent orchestration)
- RAG (retrieval augmented generation)
- ChromaDB (vector store)
- Claude 3.5 (LLM)
- FastAPI, Docker

Impact:
40% auto-resolution rate. Average resolution time dropped from 4 hours to 12 minutes.

## AI Resume Screening Agent

Problem:
Manual resume screening is slow, inconsistent, and biased.

Solution:
Multi-agent system parsing resumes against job descriptions with explainable scoring and bias detection.

Tech:
- GPT-4
- LangGraph (multi-agent workflow)
- Pinecone (vector similarity)
- PostgreSQL, FastAPI, Python

Impact:
1,500+ resumes processed in under 3 minutes. 94% accuracy vs. human reviewers. 85% reduction in screening time.
