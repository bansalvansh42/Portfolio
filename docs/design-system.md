# Design System

Theme:
* Premium AI Engineer
* Dark mode first with system preference detection
* Minimal and elegant

Typography:
* Inter
* Geist (primary, via next/font)

Spacing:
* Large whitespace
* Comfortable reading width
* Container max-width: 1280px

Components:
* Hero Section with radial gradients, title pill, resume CTA
* Skills Grid grouped by category
* Project Showcase Cards with video/demo support
* Experience Timeline
* Contact Section with social links

Animations:
* Framer Motion
* Subtle fade-in on scroll (whileInView)
* Smooth hover effects on cards and links

Inspiration:
* Vercel
* Linear
* OpenAI
* Anthropic

Requirements:
* Fully responsive
* Mobile first
* Accessibility friendly
* High Lighthouse score
* SEO optimized (sitemap, robots, OG metadata, 404 page)
