# Project Context

This project is a personal portfolio website for Vansh Bansal.

Goals:
* Showcase AI Engineering skills with quantified results
* Showcase AI Agent projects with real-world metrics
* Showcase n8n automations and enterprise impact
* Showcase AI Infrastructure and LLMOps experience
* Generate freelance and full-time Senior AI Engineer opportunities

Tech Stack:
* Next.js 16 (App Router)
* TypeScript 5
* Tailwind CSS v4
* shadcn/ui-style primitives
* Framer Motion for animations

Design Goals:
* Modern
* Clean
* Minimal
* Fast (static generation)
* Professional
* Dark Mode First with system preference

Target Audience:
* Recruiters (primary)
* Startup founders / CTOs
* AI companies
* Freelance clients

Sections:
1. Hero — Title pill, tagline with per-word hover, about, CTAs (Projects, Contact, Resume)
2. Skills — Category-grid with market-relevant tech badges (GPT-4o, Claude, Pinecone, etc.)
3. Featured Projects — 3 most relevant with quantified impacts, video/demo modals
4. Projects (full) — All 5 projects in card grid
5. Experience — Timeline with metrics per bullet
6. Contact — GitHub, LinkedIn, Email with styled cards

The design should feel similar to Linear, Vercel, Stripe, and modern AI startups.

Avoid:
* Excessive animations
* Bright gradients
* Cluttered layouts
* Generic developer portfolio templates
* Placeholder content or dead links
